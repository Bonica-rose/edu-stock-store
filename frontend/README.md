# Edu Stock&Store Frontend

Frontend application for the Edu Stock&Store - Inventory & Asset Management System.

Built using React, Vite, Redux Toolkit, Tailwind CSS, and modern frontend tooling.

---

# Tech Stack

- React.js
- Vite
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- React Hook Form
- Yup Validation
- Axios
- TanStack Table
- React Select
- React Icons
- React Hot Toast

---

# Features

- Authentication & Authorization
- Role-Based Access Control (RBAC)
- Dashboard Management
- Branch Management
- User Management
- Inventory Management
- Asset Tracking
- Vendor Management
- Purchase Management
- Responsive UI
- Reusable Form Components
- Reusable Table Components
- Global State Management
- API Integration Ready
